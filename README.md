# การลงไฟล์
1.ไห้นำไฟล์ Mod ภาษาไทย ที่เราโหลดไปมาไว้ไน 

>C:\Users\<username>\Documents\Electronic Arts\The Sims 3\Mods\Packages

  (หากไม่พบให้ทำการสร้าง Folder ชื่อ Mods >ข้างใน MOD สร้าง Folder ชื่อ Packages ด้วย )
